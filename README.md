# chessy
Полноценные шахматы на JavaScript с адаптивным CSS

Что есть:
1. Все фигуры ходят по всем правилам 
2. Подсветка возможных ходов по клику
4. Проверка на шах каждый ход
5. Пешку можно заменить на любую фигуру, когда достигнет края
6. Доска, шрифт, фигуры адаптируются под размер окна
7. Король не может ходить в опасные клетки
8. Если игнорировать шах или подставить под удар своего короля - игра окончена.

## Запуск:
1. git clone https://github.com/self-related/chessy.git
2. cd chessy
3. npx live-server

Либо сразу здесь: https://self-related.github.io/chessy/
