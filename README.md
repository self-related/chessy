# chessy
Полноценные шахматы на JavaScript с адаптивным CSS
Деплой https://self-related.github.io/chessy/

Что есть:
1. Все фигуры ходят по всем правилам 
2. Подсветка возможных ходов по клику
3. Доска, шрифт, фигуры адаптируются под размер окна
4. Проверка на шах каждый ход
5. Король не может ходить в опасные клетки
6. Если игнорировать шах или подставить под удар своего короля - игра окончена.
